## 🔬 模型技术细节

### 优化问题

```
最小化: ||G(2026) - T||_2^2

其中:   G(2026) = G(2025) + Λ·(E·x)
        Λ = diag(λ₁, ..., λ₅)  [调整速度矩阵]
        E ∈ R^{5×5}             [效率矩阵]
        x ∈ [0,1]^5             [决策变量]
        T ∈ R^5                 [目标向量]
```

### 求解方法

- **求解器**: scipy.optimize.lsq_linear (约束线性最小二乘)
- **算法**: 子空间法 (内部使用Bornacki-Golub)
- **收敛**: 全部问题均在<100迭代内收敛 ✓
- **精度**: 机器精度 (1e-10)

### 模型版本

- 版本: **拉马克离散进化模型 v1.0**
- 约束: **方案A** (完整模型，无速度截断)
- 边界: **0 ≤ x_i ≤ 1** (可视需要调整)
