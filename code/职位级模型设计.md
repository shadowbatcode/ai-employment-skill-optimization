# 职位级生态协同进化模型设计

## 1. 模型架构

### 1.1 数据层次
```
职位层级（Job Level）
├── 具体职位（如 "Specialty Trade Contractors"）
│   ├── NAICS 代码
│   ├── O*NET 代码
│   ├── 历史就业数据（2020Q1-2025Q2）
│   └── 可学习参数
└── 所属大类（STEM/Trade/Art）
```

### 1.2 每个职位的参数集

```python
职位参数 = {
    # 基础参数（从数据计算）
    'N0': 初始规模,
    'r0': 基准增长率,
    'K0': 基础承载力,

    # 可学习参数（需要优化）
    'theta': AI敏感度 [-1, 1],
    'Ac': 临界AI水平 [0, 1],
    'k': 渗透速率 [0, 10],
    'G': 技能向量 [5维],
    'cost_reduce': 成本削减比例 [0, 1],
    'epsilon': 价格弹性 [0, 2]
}
```

## 2. 参数学习流程

### 2.1 训练-验证分割
- 训练集：2020Q1 - 2023Q4（16个季度）
- 验证集：2024Q1 - 2025Q2（6个季度）

### 2.2 优化目标
最小化预测误差：
```
Loss = Σ (N_pred - N_actual)² / N_actual²
```

### 2.3 优化方法
1. **遗传算法**：全局搜索
2. **贝叶斯优化**：高效采样
3. **梯度下降**：局部优化

## 3. 时间序列预测

### 3.1 预测流程
```
1. 加载历史数据（2020Q1-2025Q2）
2. 对每个职位：
   a. 初始化参数
   b. 在训练集上优化参数
   c. 在验证集上评估
3. 使用优化后的参数预测未来（2025Q3-2030Q4）
```

### 3.2 输出
- 每个职位的未来就业规模预测
- 参数演化轨迹
- 预测置信区间

## 4. 实现模块

### 4.1 核心模块
- `job_model.py`: 职位级模型类
- `parameter_optimizer.py`: 参数优化器
- `time_series_predictor.py`: 时间序列预测器
- `data_processor.py`: 数据预处理

### 4.2 工作流
```python
# 1. 加载数据
data = DataProcessor(DATA_FILE)
jobs = data.get_all_jobs()

# 2. 对每个职位建模
for job in jobs:
    model = JobModel(job)

    # 3. 参数优化
    optimizer = ParameterOptimizer(model, train_data)
    best_params = optimizer.optimize()

    # 4. 时间序列预测
    predictor = TimeSeriesPredictor(model, best_params)
    forecast = predictor.predict(horizon=20)  # 预测20个季度
```

## 5. 关键技术点

### 5.1 参数约束
- 使用有界优化确保参数在合理范围
- 添加正则化防止过拟合

### 5.2 竞争关系
- 使用O*NET相似度矩阵
- 动态计算职位间竞争系数

### 5.3 AI事件影响
- 事件驱动的AI成熟度
- 职位特定的AI影响权重
