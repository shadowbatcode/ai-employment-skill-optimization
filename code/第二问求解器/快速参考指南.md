## 技术说明

### 模型方程
```
G(2026) = G(2025) + Λ·(E·x*)

其中：
- Λ = diag(λ₁, λ₂, λ₃, λ₄, λ₅)  [调整速度矩阵]
- E = 5×5 效率矩阵               [投入效能矩阵]
- x* = [x₁*, x₂*, x₃*, x₄*, x₅*]  [最优决策向量]
```

### 约束条件
```
最小化: ||G(2026) - T||²
约束:   0 ≤ xᵢ ≤ 1  (每项投入归一化)
求解器: scipy.optimize.lsq_linear (凸优化)
```

### 数据来源
- G(2025)：2025年实际技能评估
- T(2026)：2026年市场需求目标
- Λ：历史数据拟合的增长速度
- E：效率矩阵（跨域数据）
