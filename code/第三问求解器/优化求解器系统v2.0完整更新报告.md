## 技术实现详情

### 数学模型

**基础公式**（简化拉马克方程）:
```
G(t+1) = G(t) + Λ·(E·x)

其中:
  G(t) ∈ ℝᵐ    : 技能向量（m维）
  Λ ∈ ℝᵐˣᵐ    : 调整速度矩阵（对角）
  E ∈ ℝᵐˣᵖ    : 效率矩阵（m维技能 × p个项目）
  x ∈ [0,1]ᵖ  : 投入决策向量
```

**扩展公式**（加权评估）:
```
min Σₖ wₖ·||Gₖ(2026) - Targetₖ||²

其中 k ∈ {technical, leadership, sustainability}
```

### 优化算法

| 场景 | 算法 | 约束 |
|------|------|------|
| 无约束 | `numpy.linalg.lstsq` | 无 |
| 盒约束 | `scipy.optimize.lsq_linear` | 0 ≤ x ≤ 1 |
| 加权 + 预算 | `scipy.optimize.minimize` (SLSQP) | 0 ≤ x ≤ 1, Σx ≤ B |

### 维度自适应机制

```python
# 自动检测维度
self.m = len(self.G2025)           # 技能维度
self.p = self.E.shape[1]           # 投入项目数

# 动态构建矩阵
params0 = np.concatenate([
    original_E.flatten(),          # m×p个参数
    np.zeros(m)                    # m个调整参数
])
```

## 常见问题 (FAQ)

**Q1: 如何选择5D还是9D模型？**
A1:
- 5D模型：适合快速原型和基础分析
- 9D模型：适合全面评估和多维度决策（推荐）

**Q2: 权重方案如何影响结果？**
A2: 权重方案决定不同类型技能的相对重要性。例如：
- 综合发展 (0.4, 0.3, 0.3)：平衡发展
- 社会贡献 (0.3, 0.1, 0.6)：强调可持续性
- 技术优先 (0.6, 0.2, 0.2)：聚焦技术技能

**Q3: 参数调优有什么用？**
A3: 调优可以调整E矩阵和T向量，使优化结果更接近预期的投入分布。

**Q4: RMSE值多少算好？**
A4:
- RMSE < 0.05: 优秀
- 0.05 ≤ RMSE < 0.10: 良好
- RMSE ≥ 0.10: 需要改进

**Q5: 如何处理调参失败？**
A5:
1. 增加迭代次数 (`max_iterations`)
2. 调整目标投入分布（使其更合理）
3. 检查E矩阵是否合理

**Q6: 如何添加新的职业类别？**
A6: 在 `create_extended_data_pack()` 中添加新的数据字典。
